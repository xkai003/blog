vue项目在服务器临时开启服务

  • 步骤
    • 步骤一：进入项目文件根目录，在终端使用输入以下命令打包项目，打包后将生成一个名为dist文件夹
    npm run build
    步骤二：将生成的dist文件夹拖到WinSCP并重新命名为Xiaomi_vue 步骤三：进入WinTerm 步骤四：cd到项目根目录
    cd /etc/www/vue-app/Xiaomi_vue
    步骤五：输入以下指令使用本地开启服务
    npm install -g http-server
    http-server
    最后：使用浏览器访问您的项目：http://[您的服务器IP]:8000
    复制成功

Nginx托管静态网页

  • 步骤
    • 步骤一：打开WinSCP 步骤二：把需要挂载的网页文件夹从左边栏拖到右边栏 步骤三：打开WindTerm，在左侧选择【root文件夹】→【47.79.40.65】，输入账号密码后就可以进入 步骤四：输入以下命令进入vim编辑界面
    vim /etc/nginx/nginx.conf
    步骤五：进入vim编辑界面后，按i进入编辑模式。代码示范如下
    http {
     server {
        listen 8088;
        server_name localhost;
        root /etc/www;
        index index.html;

        location/{
         try_files $uri $uri/ = 404;
        }
     }
    }
    步骤六：编辑完成后按【esc】键后输入:wq保存退出 步骤七：输入以下命令会打开 Caddy 配置文件（这一步不用更改任何信息，但不可以省略）退出方式与步骤六一致
    vim /etc/caddy/Caddyfile
    步骤八：cd到根目录
    cd /etc/www/
    步骤九：输入以下命令启动Caddy服务
    systemctl start caddy
    步骤十：输入以下命令查看Caddy服务器的运行状态。如果看见绿色就说明启动成功，如果看见红色就说明启动失败
    systemctl status caddy

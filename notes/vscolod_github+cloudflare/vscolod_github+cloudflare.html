本地项目上传到github上并部署到cloudflare

  • 本地项目上传到github
    • 步骤一：进入github官网 步骤二：点击右上角+号 → New repository 步骤三：进入New repository页面后依次填写存储库的名称Repository name和存储库的描述Description 步骤四：点击Create repository 就创建好仓库了 步骤五：打开vscold，在项目根目录新建终端 步骤六：在终端中输入以下命令初始化 Git 仓库
    git init
    步骤七：创建并添加项目文件
    git add .
    步骤八：提交你的修改
    git commit -m "这是我的课堂管理系统"
    步骤九：关联本地仓库和远程 GitHub 仓库
    git remote add origin https://github.com/xkai003/MyClassroomManagementSystem.git
    步骤十：推送到 GitHub
    git branch -m master main
    步骤十一：推送代码到 GitHub
    git push -u origin main
    到这里就结束了。你的项目就成功上传到 GitHub 了
  • github上的项目部署到cloudflare
    • 步骤一：进入cloudflare官网 步骤二：点击Workers 和 Pages → 点击创建应用程序 步骤三：点击Pages → 点击导入现有 Git 存储库栏的开始使用按钮 步骤四：在选择一个存储库栏勾选你要部署的项目 步骤五：点击开始设置按钮 步骤六：在框架预设栏选择Vue 步骤五：点击保存并部署 到这里就部署成功了。你就可以访问你的网站了
    复制成功
